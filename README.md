## About The Project

Get Guides is a platform that connects freelance tour guides to touristsüèÉ‚Äç‚ôÇÔ∏è

When organizing a trip somewhere, finding and hiring a guide is currently the only step that requires going through some sort of professional service such as a travel agency or a travel consultantüôÖ

Get Guides aims to free travellers from that burden and allows them to directly access and interact with guidesüî•

This project was presented as part of a full-stack engineering bootcamp app showcase event.
